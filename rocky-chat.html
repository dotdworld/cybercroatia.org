Hi, I'm Rocky. Paste a process, command line, or parent-child chain and I’ll help you check whether it looks normal, suspicious, or worth deeper investigation.

Do not submit sensitive internal data, usernames, hostnames, tokens, passwords or personal information.

What can Rocky help with?

Rocky Chat is intended for quick process-behavior checks during SOC analysis. It can help explain common Windows processes, expected parent-child relationships, suspicious execution patterns, LOLBin abuse indicators and command-line behavior. Results should be used as analyst context, not as the only decision point.
